# Release Checklist

Before making this repository public on GitHub, check the following:

- [ ] No raw HBN data are included
- [ ] No subject-level restricted derivative files are included
- [ ] Hard-coded private local paths have been cleaned or documented
- [ ] A license has been selected
- [ ] The manuscript version in `paper/main.tex` is the intended public version
- [ ] The paper figures and tables match the current manuscript text
- [ ] The included result CSV/PDF/PNG files are exactly the ones cited in the paper
- [ ] Large unnecessary binaries have been removed
- [ ] The README explains the data-access restriction clearly
