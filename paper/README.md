# Paper Directory

This directory contains the curated manuscript source for the public-release package.

## Main file

- [`main.tex`](main.tex)

## Local assets

- `tables/`: tables directly included by the manuscript
- `figures/`: figures directly included by the manuscript
- `mysty.sty`, `natbib.bst`, `refLibrary.bib`: local style and bibliography files

## Compile note

A smoke compilation in this release directory succeeds with XeLaTeX. As in the working manuscript tree, a fully clean bibliography/cross-reference build may still require the usual multi-pass sequence.
