# Results Directory

This directory contains lightweight outputs that support the claims written into the manuscript.

## Matrix

- `matrix/rank_selection_log_contrast_cv5/`: the CV rank-summary table and the validation-MSE curve for the matrix log-contrast analysis
- `matrix/inference_log_contrast_rank1/`: the full loading-level inference table and the structured-loading figure for the selected rank-1 matrix model

## Tensor

- `tensor/rank_selection/`: the Tucker-rank CV summary table and the validation-MSE figure
- `tensor/inference_rank252_foldcentered/`: the full loading-level inference table and the shared/contrast figure for the fixed-rank `(2,5,2)` tensor analysis

Detailed intermediate diagnostics and process writeups are intentionally omitted from this public version.
