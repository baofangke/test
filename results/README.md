# Results Directory

This directory contains lightweight outputs that support the claims written into the manuscript.

## Matrix

- `matrix/rank_selection_log_contrast_cv5/`: 5-fold CV rank-selection outputs for the matrix log-contrast analysis
- `matrix/inference_log_contrast_rank1/`: rank-1 matrix inference outputs and figures

## Tensor

- `tensor/rank_selection/`: Tucker-rank selection summaries
- `tensor/inference_rank252_foldcentered/`: fixed-rank `(2,5,2)` tensor inference outputs and figures

These folders are intended for inspection and provenance tracking rather than as substitutes for the restricted upstream EEG data.
